# Axis VK Bot

VK-бот на Java для ролевого проекта. Работает в ЛС сообщества и в беседах VK через Bots Long Poll API, хранит профили игроков и базу знаний в SQLite, а `kie.ai` использует только для сложных вопросов и проверки постов.

## Что умеет

- Отвечает на вопросы по правилам, лору, сюжету, персонажам и локациям.
- Хранит профили игроков: имя персонажа, пол, возраст, спектр, индекс, заметку, ссылку на VK.
- Отвечает на команды вроде `Аксис мой профиль`, `Аксис мой индекс`, `Аксис какой спектр у Вильяма?`.
- Позволяет админам обновлять профили и базу знаний прямо через сообщения.
- Может проверить пост на возможные нарушения правил или лора.
- Поддерживает команду временного мута в беседе через VK API.

## Архитектура

- `SQLite` в файле `DATA_DIR/axis.db` для дешевого и простого хранения.
- Локальный поиск по базе знаний и профилям для дешевых ответов.
- `kie.ai` вызывается только для:
  - сложных вопросов, где нужно собрать ответ из нескольких записей;
  - проверки постов на нарушения.

## Переменные окружения

- `VK_GROUP_ID` - ID сообщества VK.
- `VK_GROUP_TOKEN` - токен сообщества с доступом к сообщениям и API.
- `VK_API_VERSION` - версия VK API, по умолчанию `5.199`.
- `VK_BOT_NAME` - имя обращения к боту, по умолчанию `Аксис`.
- `ADMIN_IDS` - список VK ID админов через запятую.
- `DATA_DIR` - каталог для SQLite, по умолчанию `/app/data`.
- `KIE_API_KEY` - API ключ `kie.ai`.
- `KIE_BASE_URL` - базовый URL `kie.ai`, по умолчанию `https://api.kie.ai`.
- `KIE_MODEL` - путь модели, по умолчанию `gemini-3-flash`.
- `AI_ENABLED` - включить ИИ-функции (`true` или `false`).

## Настройка VK

1. Включите сообщения сообщества.
2. Включите `Long Poll API`.
3. Разрешите событие `message_new`.
4. Создайте токен сообщества с правами на сообщения и управление беседами.
5. Добавьте бота в нужную беседу.

## Команды пользователей

- `Аксис help`
- `Аксис мой профиль`
- `Аксис мой индекс`
- `Аксис мой спектр`
- `Аксис профиль Вильям`
- `Аксис какой спектр у Вильяма?`
- `Аксис какой индекс у id123456?`
- `Аксис проверь пост: <текст>`

В личных сообщениях сообщества можно писать и без префикса `Аксис`.

## Команды администратора

Все админ-команды используют формат `;` и `ключ=значение`, чтобы их было проще разбирать без лишней магии.

### Профили

Создать или обновить профиль:

```text
Аксис админ профиль set ; пользователь=id123456 ; имя=Вильям ; пол=мужской ; возраст=24 ; спектр=Алый ; индекс=IX-14 ; заметка=Старший дипломат
```

Частично обновить профиль:

```text
Аксис админ профиль update ; пользователь=https://vk.com/id123456 ; спектр=Лазурный ; индекс=IX-18
```

Показать профиль:

```text
Аксис админ профиль get ; пользователь=id123456
```

### База знаний

Добавить запись:

```text
Аксис админ знание add ; категория=правила ; заголовок=Размер постов актива ; ключи=пост,актив,размер ; текст=Минимум 2000 символов без пробелов.
```

Удалить запись:

```text
Аксис админ знание delete ; id=1
```

### Модерация

Выдать мут:

```text
Аксис админ мут ; пользователь=id123456 ; время=30м ; причина=Флуд
```

Снять мут:

```text
Аксис админ размут ; пользователь=id123456
```

## Локальный запуск

```bash
mvn clean package
VK_GROUP_ID=123456789 \
VK_GROUP_TOKEN=vk1.a.your_group_token \
ADMIN_IDS=11111111 \
KIE_API_KEY=your_kie_api_key \
java -jar target/axis-vk-bot-1.0.0.jar
```

## Docker

Сборка:

```bash
docker build -t axis-vk-bot .
```

Запуск:

```bash
docker run -d --name axis-vk-bot --restart unless-stopped \
  -e VK_GROUP_ID=123456789 \
  -e VK_GROUP_TOKEN=vk1.a.your_group_token \
  -e ADMIN_IDS=11111111,22222222 \
  -e KIE_API_KEY=your_kie_api_key \
  -v "$(pwd)/data:/app/data" \
  axis-vk-bot
```

## Ограничения

- Для мута в беседе у сообщества должны быть нужные права, а метод VK должен быть доступен для текущего типа беседы.
- Если `kie.ai` выключен или не настроен, бот продолжит работать на локальной базе знаний и профилях, но без умной сводки и проверки постов.
